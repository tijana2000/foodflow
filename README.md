# FoodFlow
FoodFlow is an in-progress food delivery web application.

## Current Status
This project is currently in-progress.
Implemented so far:
- Microservices backend structure
- Eureka Discovery Server
- API Gateway
- Restaurant Service
- User Management Service
- Order Service
- JWT authentication
- Password hashing with BCrypt
- MySQL database integration

## Planned:
- Payment Service
- Reporting Service
- React frontend
- Docker setup
- Improved documentation
  
## Tech Stack
- Java 21
- Spring Boot
- Spring Cloud Gateway
- Eureka Discovery Client/Server
- Spring Security
- JWT
- MySQL
- Maven
- React planned
  
## Architecture
